常用命令
1.导出包：`pip freeze > requirements.txt`

2.安装包：`pip install -r requirements.txt`

3.删除包：`pip uninstall -r requirements.txt -y`

4.ubuntu系统下安装包：`sudo pip3 install -r requirements.txt --user`
